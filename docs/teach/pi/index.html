前置知识

开始之前，确保你具备以下基础

💻 技术要求

  • TypeScript 熟练（中级以上）
  • Node.js 与 npm 使用经验
  • 对 LLM API 有基本认知（prompt、token、streaming）
  • 熟悉终端命令行操作

📚 推荐工具

  • Node.js 18+ 与 npm
  • VS Code 或类似编辑器
  • 一个 LLM API 密钥（可选）
  • Git 基础操作能力
