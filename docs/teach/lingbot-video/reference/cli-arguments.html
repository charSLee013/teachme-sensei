CLI Arguments Reference

本页按行为而非 argparse 顺序组织公开推理参数。

输入与输出

参数作用
`--model_dir`LingBot-Video 模型根目录，必填。
`--mode``t2i`、`t2v` 或 `ti2v`。
`--prompt_json`推荐输入，读取 structured caption 和 optional duration。
`--prompt`直接 prompt 字符串，通常只用于调试。
`--image`TI2V 首帧路径；TI2V refiner 必需。
`--output`base 输出路径；T2I 为图片，视频模式为 MP4。

Prompt 与 Negative

参数作用
`--negative_prompt`手写 negative prompt。
`--negative_prompt_json`读取 auto negative 输出；与 `--negative_prompt` 互斥。

几何与采样

参数作用
`--height`, `--width`输出几何；需要满足 Pipeline input checks。
`--resolution`, `--ratio`使用预设 bucket；必须同时提供。
`--duration`视频时长；如果 prompt JSON 提供 duration，runner 会使用它。
`--num_frames`手写帧数；当前代码中会被 prompt JSON duration 覆盖。
`--steps`, `--guidance_scale`, `--shift`, `--seed`, `--fps`采样步数、CFG、flow shift、随机种子和视频帧率。

精度与组件加载

参数作用
`--default_dtype`未被组件级参数覆盖时的默认 dtype。
`--transformer_dtype`DiT transformer 主体加载 dtype。
`--text_encoder_dtype`Qwen3-VL text/VLM encoder 相关组件 dtype。
`--vae_dtype`base VAE dtype；代码会检查实际 VAE dtype 是否匹配请求值。
`--transformer_subfolder`模型目录内 transformer 子目录名，默认 `transformer`。
`--diffusers_attn_backend`设置 `DIFFUSERS_ATTN_BACKEND`，常见脚本默认 `_native_flash`。
`--allow_tf32`, `--no-allow_tf32`控制 CUDA matmul TF32；默认允许。
`--quiet_progress`关闭加载日志与 denoising 进度输出，同时设置 `LINGBOT_QUIET_PROGRESS`。

Backend 与并行

参数作用
`--backend``diffusers` 或 `sglang`。
`--batch_cfg`批量计算 CFG 两支。
`--cfg_parallel_degree`跨 rank 拆分 CFG 分支。
`--context_parallel_degree`跨 rank 切分长序列。
`--context_parallel_ulysses_anything`传给 context parallel plan 的兼容开关，用于更宽松地启用 Ulysses-style 切分。
`--enable_fsdp_inference`对 transformer 做 inference-time FSDP sharding。
`--null_cond_clone_zero`非 TI2V 模式下控制 null condition 的 clone-zero 策略。
`--reuse_condition_features`缓存并复用 prompt condition features，尤其用于 refiner 或多阶段调用。

Refiner

参数作用
`--run_refiner`启用视频 refiner。
`--refiner_model_dir`非标准模型布局时覆盖 refiner 根目录。
`--refiner_transformer_subfolder`refiner transformer 子目录名，默认 `refiner`。
`--refiner_output`refiner 输出路径。
`--refiner_height`, `--refiner_width`refiner 目标分辨率。
`--refiner_steps`, `--refiner_guidance_scale`, `--refiner_shift`refiner 步数、CFG 强度与 flow shift。
`--refiner_t_thresh`, `--refiner_sigma_tail_steps`refiner 低噪声区间和尾部 sigma 步数配置。
`--refiner_fps`, `--refiner_sample_fps`refiner 输出帧率与从 base 视频抽样的帧率。
`--refiner_max_video_frames`限制进入 refiner 的最大视频帧数。
`--refiner_vae_dtype`refiner VAE dtype。
`--refiner_batch_cfg`refiner 内批量计算 CFG 两支；与 `--cfg_parallel_degree > 1` 互斥。
`--refiner_no_null_cond_clone_zero`关闭 refiner null condition clone-zero 策略。
`--refiner_offload_vae_during_denoise`denoising 期间卸载 refiner VAE，以换取显存余量。
