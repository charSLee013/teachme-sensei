File Map

本页按概念映射文件，而不是复述目录树。

概念主要文件阅读目的
公开 CLI`scripts/inference.py`理解入口为何很薄，以及默认 backend 如何注入。
Backend 解析`lingbot_video/inference_backend.py`理解 diffusers/sglang/fallback 和 negative JSON 加载。
推理编排`lingbot_video/runner.py`理解参数、模型加载、并行、refiner 和保存输出。
模型路径`lingbot_video/model_paths.py`理解 component 目录和 refiner root 解析。
基础 Pipeline`lingbot_video/pipeline_lingbot_video.py`理解 prompt encode、latent、scheduler、CFG、VAE decode。
TI2V Pipeline`lingbot_video/pipeline_lingbot_video_i2v.py`理解首帧图像和 clean latent prefix。
Transformer`lingbot_video/transformer_lingbot_video.py`理解 single-stream DiT、3D RoPE、MoE。
Scheduler`lingbot_video/scheduling_flow_unipc.py`理解 FlowUniPC timesteps 和 step 接口。
Utils`lingbot_video/utils.py`理解帧数、refiner sigmas、视频读取、batch CFG。
FSDP`lingbot_video/fsdp_inference.py`理解 inference-time sharding。
SGLang adapter`lingbot_video/native_backend.py`、`sglang_moe_shim.py`理解 native wrapper 和 MoE fused kernel shim。
Prompt 重写`rewriter/inference.py`、`rewriter/rewriter_core.py`、`rewriter/system_prompts.py`理解两阶段 rewrite。
Auto Negative`rewriter/auto_negative.py`理解 delete-only negative pruning。
