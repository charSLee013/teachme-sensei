Model and Runtime Card

本页将 Ideogram 4 推理包描述为可运行模型接口，而不是概念图。字段覆盖模型身份、变体、输入输出、运行时、许可、访问、安全与限制。

Abstract

Claim

该仓库提供 Ideogram 4 的本地推理接口：输入为 caption string 和图像尺寸/采样参数，输出为 `PIL.Image`。模型权重通过 Hugging Face gated repositories 获取，CLI 暴露 nf4 和 fp8 两类变体。

Evidence. `run_inference.py` 的 `QUANTIZATION_REPOS`、`Ideogram4PipelineConfig`、README model zoo 和 license files。

Boundary. 未认证环境无法读取 Ideogram HF raw model cards；本页只使用本地源码、README 和可访问官方页面中的信息。

Safety, Access, License

Access

README states model weights are gated on Hugging Face. The course environment observed 401 Unauthorized when fetching raw HF README URLs without credentials.

License

Source code is Apache-2.0. Model weights use the Ideogram Non-Commercial Model Agreement included under `model_licenses/`.

Safety

Hive moderation is integrated but key-dependent. Missing keys produce CLI warnings, not blocking moderation.

Known Limits

  • Training data, optimizer, compute and duration are not exposed by this inference repository.
  • Real image quality, latency and memory behavior require authenticated weights and hardware-specific measurement.
  • Prompt safety and output safety depend on configured Hive keys or equivalent deployment controls.
