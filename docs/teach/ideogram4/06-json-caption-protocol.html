Prompt protocol / structured condition

JSON Caption Protocol

Ideogram 4 的 prompt 设计重点是把画面条件结构化：整体描述、风格、媒介、颜色、背景、对象、文字和可选位置约束被组织成 JSON caption。magic prompt 在 CLI 默认路径中承担 plain text 到 structured caption 的编译器角色。

Abstract

本章解释的是这个仓库里真正可执行的 prompt 协议：JSON caption 的字段语义、约束关系，以及为什么 magic prompt 会把自然语言先编译成结构化条件。

Claim and Evidence

Claim

The repository treats structured JSON captions as the high-control prompt protocol and includes a verifier for schema, key order, bounding boxes, and color palette constraints.

Evidence. `docs/prompting.md`, `caption_verifier.py`, `magic_prompt.py`, CLI magic-prompt default behavior.

Boundary. Comparisons with other prompting paradigms in this page are lineage context for understanding design tradeoffs, not claims about Ideogram's private training or competitors' internals.

Schema Semantics

FieldCondition representedWhy it matters
`high_level_description`Global semantic intent.Gives the model a concise whole-image target.
`style_description.aesthetics`Visual mood and style adjectives.Separates style from object content.
`lighting`Illumination condition.Controls scene rendering independent of object list.
`photo` / `art_style`Photographic or non-photographic rendering mode.Enforces mutually exclusive style branch in verifier.
`color_palette`Global or per-element colors.Makes palette control explicit instead of implicit prose.
`background`Scene environment.Separates context from foreground elements.
`elements[]`Objects or text instances, optional bbox.Gives element-level composition and typography targets.

Magic Prompt as Compiler

In the CLI default path, a plain prompt is not sent directly to the model. It is expanded through a registered magic prompt backend into a minified JSON caption. This makes magic prompt analogous to a compiler from casual intent to structured model condition.

Failure condition

If magic prompt is enabled but no API key is available, CLI exits before model loading. If magic prompt is disabled, the caller must provide a prompt acceptable to the verifier or use warning mode.
