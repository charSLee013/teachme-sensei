Abstract

本章直接把 `run_inference.py` 拆成接口契约：参数默认值、行为影响、失败模式和相互作用都在表中说明，避免读者把 CLI 误读成简单的 prompt 转发层。

Claim and Evidence

Claim

CLI 是完整推理路径的用户入口，但它不只是转发 prompt。它决定设备、量化权重、prompt expansion、安全筛查、采样 preset、seed 和输出位置。

Evidence. `argparse` definitions and control branches in `run_inference.py`.

Boundary. 该页描述 CLI 行为；`Ideogram4Pipeline.__call__` 的内部采样机制在 sampling 章节展开。

Argument Contract

ArgumentDefaultEffectFailure mode / interaction
`--prompt`requiredInput text. With magic prompt on, it is expanded before model inference.Missing value makes argparse fail.
`--output``output.png`File path for saving first generated image.Filesystem write errors happen after generation.
`--height`1024Contributes to image token grid and schedule resolution.Pipeline requires divisibility by 16.
`--width`1024Contributes to image token grid and aspect ratio for magic prompt.Pipeline requires divisibility by 16.
`--sampler-preset``V4_QUALITY_48`Sets `num_steps`, guidance schedule, `mu`, `std`.Choices restricted to `PRESETS` keys.
`--seed`0Seeds latent noise generator for reproducible generation.Same seed does not guarantee same output across changed code/hardware/deps.
`--device`CUDA, else MPS, else CPUDevice for pipeline components and sampling tensors.nf4 path rejects non-CUDA.
`--quantization`CUDA: nf4; else fp8Selects Hugging Face repo and quantized loading path.Gated repo access required; incompatible device can fail.
`--magic-prompt` / `--no-magic-prompt`onControls plain prompt expansion into structured caption.If on and no key exists, CLI exits before loading model.
`--magic-prompt-model``ideogram-4-v1`Selects Ideogram hosted API or OpenRouter Claude implementation.Backend requires matching API key and network availability.
`--magic-prompt-key``MAGIC_PROMPT_API_KEY` or `IDEOGRAM_API_KEY`Authenticates prompt expansion service.Required unless `--no-magic-prompt`.
`--warn-on-caption-issues`falseTurns caption verifier failures into warnings.Can allow out-of-schema prompts to reach model.
`--hive-text-key``HIVE_TEXT_MODERATION_KEY`Enables prompt moderation before generation.Missing key disables prompt screening with warning.
`--hive-visual-key``HIVE_VISUAL_MODERATION_KEY`Enables generated image moderation before saving.Missing key disables output screening with warning.

Operational Sequence

parse args -> optional Hive text moderation -> optional magic prompt expansion -> select sampler preset -> load Ideogram4Pipeline from selected quantized repo -> pipe(prompt, height, width, preset params, seed) -> optional Hive visual moderation -> save first PIL image

This sequence is an interface contract: a failure before pipeline load avoids downloading weights; a failure after generation may still consume inference resources.
