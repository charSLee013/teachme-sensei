核心结论

结论依据类型边界
该仓库是 Ideogram 4 的推理工具包。源码事实 + 本地文档。不覆盖私有训练系统。
pipeline 会把文本特征和图像 latent token 打包进共享序列。来自 `pipeline_ideogram4.py` 的源码事实。解释推理布局，不解释训练数据组成。
模型核心是 34 层单流 Transformer velocity predictor。来自 `modeling_ideogram4.py` 和本地文档的源码事实。架构名称以仓库源码和文档为准。
采样会组合文本条件 velocity 和 image-only baseline velocity。来自 pipeline sampling loop 的源码事实。质量影响需要另行实验，不在本课程中声称。
结构化 JSON caption 是高控制提示词协议。本地 prompting 文档和 verifier 实现。与其他提示词范式的比较仅用于背景定位。
公开资料没有给出完整训练细节。源码缺失 + 公开资料边界。未公开不等于不存在，只表示当前材料无法确认。
