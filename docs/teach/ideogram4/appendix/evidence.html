Core Claims

ClaimEvidence typeBoundary
The repository is an inference toolkit for Ideogram 4.Source fact + local documentation.Does not include private training system.
The pipeline packs text features and image latent tokens into a shared sequence.Source fact from `pipeline_ideogram4.py`.Explains inference layout, not training data composition.
The model core is a 34-layer single-stream Transformer velocity predictor.Source fact from `modeling_ideogram4.py` and local docs.Architecture names follow repo source/docs.
Sampling combines conditional and image-only unconditional velocity predictions.Source fact from pipeline sampling loop.Quality implications require generation experiments not performed here.
Structured JSON captions are the expected high-control prompt protocol.Local prompting docs and verifier implementation.Comparisons with other prompting paradigms are lineage context.
Training details beyond public high-level claims remain unknown in this repository.Source absence + public docs boundary.Unknown means not confirmed here, not necessarily nonexistent.

Evidence Language Standard

Use

“The implementation defines…” for source facts.

Use

“Public materials state…” for release claims.

Use

“This report reconstructs…” for teaching explanations.

Avoid

Speculating about training data, compute, optimizer, or private methods.
