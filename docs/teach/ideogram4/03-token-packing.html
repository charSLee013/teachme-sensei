Representation layer / sequence construction

Token Packing Mechanism

Ideogram 4 的推理输入不是单一文本向量。Pipeline 构造一个 packed sequence：左侧是 Qwen3-VL 文本 token 对应的 hidden features，右侧是 image latent tokens。并行的 position、segment 和 indicator 轨道定义了每个 token 在 Transformer 中的角色。

Abstract

本章围绕 packed sequence 解释推理输入的真实结构：文本特征、图像 latent token、位置轨道、段轨道和 indicator 轨道如何共同决定 Transformer 看到的序列。

Claim and Evidence

Claim

`_build_inputs` constructs a packed sequence with text token slots followed by image latent token slots. It also constructs `position_ids`, `segment_ids`, and `indicator` tensors used by the Transformer.

Evidence. `pipeline_ideogram4.py::_tokenize`, `_build_inputs`, `_encode_text`; constants `LLM_TOKEN_INDICATOR`, `OUTPUT_IMAGE_INDICATOR`, `IMAGE_POSITION_OFFSET`.

Boundary. This describes inference tensor construction. It does not assert how training batches were built internally.

Definitions

ObjectShape / valueRole
`token_ids``(B, total_seq_len)`Qwen text token ids; image slots stay zero.
`position_ids``(B, total_seq_len, 3)`MRoPE coordinates for text and image tokens.
`segment_ids``(B, total_seq_len)`Block-diagonal attention grouping and padding separation.
`indicator``(B, total_seq_len)`Distinguishes LLM tokens from output image tokens.
`num_image_tokens``height/16 * width/16`Image latent token count after VAE compression and 2x2 patching.

Text fallback

The sequence is ordered as text tokens first and image tokens second. Text positions are `[pos, pos, pos]`. Image positions are `[0, h, w] + 65536`. The indicator track marks text positions with `LLM_TOKEN_INDICATOR` and image positions with `OUTPUT_IMAGE_INDICATOR`.

3D Lattice Interpretation

The 3D view is intentionally schematic: it maps sequence roles into a spatial metaphor. Text tokens form a raised rail; image tokens form an H x W plane. Rotate the lattice to separate sequence role, grid geometry, and token metadata.

3D fallback

The raised blue rail represents text positions carrying Qwen-derived features. The red plane represents image latent tokens with grid coordinates. This is a schematic projection for explanation; it is not a literal embedding space or learned geometry.

Mechanism Consequences

  • Increasing resolution increases image token count quadratically.
  • Longer prompts increase text sequence length but do not change image grid size.
  • Text/image role is not inferred from position alone; `indicator` explicitly controls masking and embeddings.
  • Padding remains separated by segment ids so it does not become part of the real sample sequence.
