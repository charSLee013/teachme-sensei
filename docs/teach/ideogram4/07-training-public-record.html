Claim and Evidence

Claim

Training information for this repository should be read as layered public record: public release claims, source-supported inference behavior, course reconstruction, and unknown implementation details.

Evidence. README, prompting documentation, safety documentation, and local inference source.

Boundary. Dataset composition, optimizer, compute, duration, and detailed post-training methods are not confirmed by this checkout.
