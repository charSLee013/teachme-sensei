Lesson 19 / Related Work

Related work 与研究空白

Related work 为研究问题建立坐标系，说明已有方法解决了什么以及仍有哪些空白；引用数量服务于这条论证。

前置：paper cards产物：comparison map实验：分类与删文

Remember now

每篇相关工作都要改变读者对你的问题的理解；如果删掉它，论证没有任何变化，那它可能只是装饰性引用。

直觉：画地图，不建墓碑

把文献按问题和机制排布，年份用于标记发展顺序。对 Krea 2 主线，可用“表示空间、条件融合、生成路径、采样调度、评测/系统”五个轴。

representation -> VAE / latent modeling conditioning -> text encoder / cross-modal fusion generation objective -> diffusion / flow matching sampling -> Euler / guidance / distillation evaluation/system -> quality / alignment / latency / memory

公式：研究空白是差集

将已有能力集合记为 Prior，你的目标能力为 Need：

Gap = Need - (Prior ∩ Evidence)

这个差集表示现有证据尚未回答一个明确问题。毕业主线关注公开 Krea 2 推荐设置在不同 resolution、CFG、Raw/Turbo 下的质量效率边界和失败条件，并要求可复现的系统测量。

代码：用结构化表替代段落堆砌

works = [
  {"axis": "sampling", "method": "flow matching", "evidence": "time grid"},
  {"axis": "conditioning", "method": "multi-layer Qwen states", "evidence": "config"},
  {"axis": "system", "method": "CUDNN + compile", "evidence": "hardware test"},
]
grouped = group_by(works, key="axis")

论文写作时表格不一定原样出现，但作者需要先有这样的内部地图，才能避免重复工作和无关引用。

实验：删掉一半文献

  1. 为每篇文献写它支撑的唯一 claim。
  2. 删掉不能改变研究问题的引用。
  3. 给剩余文献标记 primary、official docs、experiment 或 context。
  4. 检查每个关键背景 claim 至少有一个强来源。

这是训练选择能力，不是鼓励少引用；引用应该有功能和范围。

Evidence and scope

Source：Krea 2 official sources 与论文来源笔记。

Teaching use：将 Krea 2 的工程观察转成一个不夸大 novelty 的研究空白。

Scope：“公开设置缺少系统测量”需要通过文献检索和实验进一步确认，课程不会把它未经核查写成绝对事实。

Retrieval practice

  1. 为什么“没人做过”是危险的 related-work 句子？
  2. 怎样判断一篇工作与毕业题目真正相关？
  3. 研究空白与研究贡献有什么区别？
分层答案

标准答案：你无法仅凭有限检索证明没人做过。相关性应看问题、变量、数据/模型、指标和证据是否重叠。gap 是尚未被充分回答的问题；contribution 是你实际提供的方法、测量、数据或解释，必须由实验支撑。

常见误解：换一个模型名称不自动形成新颖性。
