Lesson 27 / Capstone

毕业论文包

现在要把一个研究问题、一次受控实验和一组有限结论组织成完整论证；前面页面只提供可追溯的素材。

前置：全课程产物：论文包验收：rubric + 答辩

Remember now

毕业标准是证据链完整；结果的视觉吸引力无法替代以下要求：问题清楚、协议可复现、比较公平、指标有范围、负结果被记录、claim 不超过 evidence。

直觉：把实验日志变成论文

实验日志按时间排列，论文按读者问题排列。先从所有记录中提炼主线，再选择能支持主线的表、图和失败样本；不要反过来根据已有图片编故事。

research question -> protocol + manifest -> baseline table -> ablation table -> qualitative/error figures -> findings + uncertainty -> claims + limitations -> paper + rebuttal + artifacts

公式：毕业论文的最低证据闭环

对每条主要结论建立：

Claim -> Measurement -> Dataset/Prompt/Seed -> Result -> Scope

如果缺任一项，标记为“需要补证据”或降低为观察性描述。毕业论文不允许用“模型感觉更好”作为最终 claim。

实验：最终验收 rubric

维度通过标准
机制理解能用直觉、公式、代码、实验四层解释完整 pipeline。
实验设计有固定项、变化项、seed、prompt、指标和失败规则。
证据每条主 claim 有表/图/原始记录和 scope。
复现另一个人能按 manifest 重跑或明确知道不能重跑的原因。
写作中英文摘要、方法、实验、限制和 caption 结构完整。
答辩能在 90 秒内解释问题、方法、发现、限制。

Evidence and scope

Source：课程 mission、第 15-26 课的实验/写作 contract。

Teaching use：把分散练习汇总为一项可观察的研究能力验收。

Scope：通过 rubric 不代表论文必然被会议录用；它只证明课程设定下的研究工作达到可审阅状态。

Retrieval practice

  1. 怎样判断一条 claim 超过了 evidence？
  2. 为什么论文包必须同时包含 raw results 和 PDF？
  3. 如果最终没有发现显著差异，毕业项目还成立吗？
分层答案

标准答案：检查 claim 的对象、条件、指标和范围是否都在记录中；缺少条件或把局部结果写成普遍规律就是超出。raw results 允许复算统计和检查选择偏差。没有显著差异仍可回答“在本协议下没有观察到差异”，并提供不确定性、设计分析和后续方向。

常见误解：研究成果可以可靠地排除一个直觉；找到提升只是可能结果之一。
