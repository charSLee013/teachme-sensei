Anima 公式与符号参考

Rectified Flow / Flow Matching

x_t = (1 - t) x_0 + t epsilon
v* = epsilon - x_0
L = E[ || v_theta(x_t, t, c) - (epsilon - x_0) ||^2 ]

用途：

  • 说明 flow-family 目标与 DDPM 噪声路径的差异。

DiT token 成本

N = latent image token count
d = hidden width

self-attention ~ O(N^2 d)
MLP / projection ~ O(N d^2)

用途：

  • 说明为什么分辨率提高后，token 成本会快速主导预算。

FID

FID = ||mu_r - mu_g||^2 + Tr(Sigma_r + Sigma_g - 2 (Sigma_r Sigma_g)^(1/2))

用途：

  • 衡量分布距离，但在动漫域存在 domain shift。

CLIP Score

CLIPScore = E[ cosine( f_img(x), f_text(c) ) ]

用途：

  • 衡量通用文本对齐，不足以完整覆盖角色一致性与尾部分布。

Strong Model Collapse 提示项

Delta Error ~ p_2^2 * tr(Delta)

用途：

  • 说明少量 synthetic contamination 也可能带来可测影响。

学习提醒

  • 公式在这里是 reference，不是结论。
  • 任何公式要进入论文主结果，都还需要和具体实验读数绑定。
