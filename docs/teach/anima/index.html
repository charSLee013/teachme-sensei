你会得到什么

读完主线后，应当能够回答这五个问题：

  • Anima 的公开架构事实是什么，哪些只是重建解释。
  • 为什么 2B 级 DiT 可能构成一个动漫领域适配的 operating point。
  • 为什么 zero synthetic data、non-anime art mixture、text-side policy 会同时影响训练结果。
  • 为什么 prompt 机制不能再按旧的 T5-XXL + LM Adapter 叙述去理解。
  • 为什么动漫模型评估不能只看 FID。

公开事实与解释边界

这套材料默认使用以下边界：

  • 已公开确认：可直接由模型卡、Diffusers config 或公开仓库文件支持。
  • 解释性重构：基于公开配置与相关文献，对机制进行的教学性整理。
  • 尚未公开确认：原始训练步数、真实 optimizer、完整过滤阈值、原始 curriculum 等。

纠错说明
