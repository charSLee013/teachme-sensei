Root Entry

Courses, not just files.

This page is the public entry for the teaching packages published under /teach/*/. The point is not to browse a folder tree. The point is to tell apart course types quickly, pick the right reading path, and reach the proof layer when you want to check how rigorous a package is.

Quick Paths

  1. Start Course: go straight into the student-facing course home.
  2. Structure: use the course map, reading order, or module index first.
  3. Evidence / Proof: open the source matrix, evidence appendix, or validation logs before committing time.
