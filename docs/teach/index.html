课程 Catalog

这里有两类不同的学习对象。 Two different learning objects live here.

Pi 是面向工程贡献者的代码仓库课程；Ideogram 4 是面向技术读者的推理系统课程。两者不是同一门课的两个章节，而是服务不同学习任务的独立教学包。

Pi is a repo course for engineering contributors. Ideogram 4 is a technical course for understanding an inference system. They are separate course packages, not two chapters of one curriculum.
