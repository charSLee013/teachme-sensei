课程 Catalog

这里有五类不同的学习对象。 Five different learning objects live here.

Pi 是面向工程贡献者的代码仓库课程；Ideogram 4 是面向技术读者的推理系统课程；Anima 是面向研究复现与论文准备的模型方法课程；FLUX.2 是面向 repo 到训练迁移决策的研究课程；LingBot-Video 是面向视频生成推理系统和论文机制的 repo + deep-paper 课程。五者服务不同学习任务，彼此独立。

Pi is a repo course for engineering contributors. Ideogram 4 is a technical course for an inference system. Anima is a model-method course for research reproduction and paper preparation. FLUX.2 is a research course for moving from repo understanding toward training decisions. LingBot-Video is a repo and deep-paper course for a video-generation inference system. They are independent course packages.
