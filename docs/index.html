  1. Start course: open the learner-facing course home.
  2. Inspect structure: use the map, module index, or chapter sequence.
  3. Check boundaries: use learner-facing evidence notes. Repository support materials remain in the control plane.
课程 Catalog

这里有六类不同的学习对象。 Six different learning objects live here.

Pi 面向工程贡献；Ideogram 4 面向推理系统；Anima 面向研究复现与论文准备；FLUX.2 面向 repo 到训练迁移决策；Krea 2 面向多模态生成代码、实验方法与 ACM MM 写作；LingBot-Video 面向视频生成推理系统和论文机制。六门课程各自服务明确的学习任务。

Pi covers engineering contribution. Ideogram 4 covers an inference system. Anima covers research reproduction and paper preparation. FLUX.2 covers repo-to-training decisions. Krea 2 connects multimodal-generation code, experimental method, and ACM MM writing. LingBot-Video covers a video-generation inference system and paper mechanisms. Each course serves a distinct learning task.
